Forma correta de usar negrito e italico

Nessa frase usamos um termo em negrito, usando a tag <b>(não semântica).

Nesta frase temos um termo em destaque usando a forma correta, <strong>.

Italico / Ênfase

Nessa forma temos um termo em italico usando a tag <i>.

Nesta frase temos um termo em Ênfase usando a tag <EM>.

Texto marcado

Podemos criar um texto marcado usando a tag <MARK> para ficar como marca texto, literalmente.

Aqui vamos criar um texto grande e um texto pequeno, usando forma não semântica e semântica. Usamos as tags <big> e <small>.

Texto deletado

Vamos marcar um texto como excluido para indicar que ele deve ser lido mas desconsiderado! Pra isso usamos a tag <del>

texto inserido

Podemos marcar um texto como inserido para dar uma Ênfase e indicar que ele foi adicionado depois! Usando a tag <ins>.

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup>.

Texto subescrito

Para inserir coisas do tipo H20 usando a tag <sub>.
